# SFML Chess Game

This project is a simple chess game developed in C++ using the SFML library. It was built as part of my learning process to understand how object-oriented programming concepts can be applied in a real-world application.

## Features

- Chess board interface with all standard pieces
- Basic movement logic for each piece
- Checkmate detection
- Option to restart the game

## Technologies Used

- C++
- SFML (Simple and Fast Multimedia Library)
- Object-Oriented Programming concepts

## What I Learned

While working on this project, I focused on improving my understanding of:

- Structuring programs using classes
- Applying concepts like encapsulation, inheritance, and abstraction
- Handling user input and game loops
- Rendering graphics using SFML
- Debugging and refining logic

This project helped me move beyond basic programs and understand how larger applications are built.

## How to Run

1. Install the SFML library on your system
2. Compile the code using a C++ compiler (such as g++)
3. Link the SFML libraries properly during compilation
4. Run the generated executable
